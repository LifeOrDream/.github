# LifeOrDream ⛩️

We build **on-chain game economies** and the tooling to run them.  
Focus areas: DeFi primitives, programmable NFTs, and AI-assisted creation on **Solana** and **Sui**.

 Infrastructure for **player-owned game economies**.                                                                                                                                                                                                                                                                      
  We build the economic layer that turns on-chain games into **self-sustaining economies** — programmable tokens with enforced tax policies, tax-aware AMMs, DNA-driven NFTs, and staking systems where every player action feeds back into the system.

## Products

 ### **<a href="https://minebtc.fun/" target="_blank">MineBTC</a>** — Solana

  Faction warfare game with an integrated DeFi economy. Players bet SOL across 12 factions in 60-second rounds, mine DogeBTC through gameplay, and stake for dual rewards.

  - **DogeBTC** — Token-2022 with enforced 1% transfer tax routed to burn, faction treasury, and NFT floor sweep
  - **Staking** — Stake DogeBTC, LP tokens, or Doge NFTs for up to 45x hashpower multiplier (lockup + NFT boost)
  - **Bitcoin Doges** — 24,690 Metaplex Core NFTs with 256-bit on-chain DNA encoding faction, evolution, appearance, and power traits
  - **Dynamic emissions** — 4-hour TWAP oracle adjusts token emission rate based on price movement, with excess SOL directed to protocol-owned liquidity
  - **Provably fair** — Commit-reveal randomness with on-chain verification

### **<a href="https://honeyplay.ai/" target="_blank">HoneyPlay</a>** — Sui

  Financial operating system for on-chain game economies. A modular stack that any game can plug into instead of rebuilding tokenomics from scratch.

  - **HoneyTaxConfig** — Policy-enforced token tax primitive splitting every action into burn, treasury, creator, and protocol streams — enforced at the Move level, no wrapping needed
  - **Tax-aware AMM** — Custom pools (constant product, stable swap, curved) with native tax handling and dynamic fees based on market cap and liquidity
  - **NFT Marketplace** — Floor sweep automation, trait-based bidding, and capital-efficient orders backed by ggSUI yield
  - **HoneyOracle** — Feedback-controlled monetary policy adjusting HONEY emission based on real-time TWAP, with automatic dampening during sell pressure
  - **ggSUI** — Liquid staking integration channeling Sui validator yield into bids, liquidity, and treasury reserves
  - **HoneyJar** — Soulbound reward ledger distributing HONEY proportional to economic contribution across all modules
